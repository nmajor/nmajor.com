# Raw source capture — Microsoft cancels Claude Code for cost certainty (fetched 2026-07-07)

## Jerusalem Post — "Microsoft reduces its reliance on Claude"
https://www.jpost.com/business-and-innovation/article-900131
- Dec 2025: Microsoft rolled Anthropic's Claude Code to thousands of employees across its Experiences & Devices division (Windows, Microsoft 365, Outlook, Teams, Surface).
- <6 months later: started cancelling most of those licenses, redirecting engineers to its own GitHub Copilot CLI ahead of a June 30, 2026 deadline.
- "The tool wasn't removed because engineers disliked it. Quite the opposite. It was removed because they used it too much."
- Root cause = pricing. Frontier AI tools charge by token usage; every prompt/review/debug adds to the bill. Per-engineer costs reportedly $500-$2,000/month. US AI software prices climbed 20-37% over the past year.
- One AI consultant told Axios a client spent half a billion dollars in a single month after failing to cap employee Claude usage.
- Microsoft did not abandon Claude entirely: Anthropic models remain available via Copilot CLI and Microsoft's Foundry partnership (investment up to $5B in Anthropic). "What ended was the expensive product layer sitting on top of the models."

## Developers Digest — "Enterprise AI Coding Budget Blowouts: What Uber and Microsoft Teach Us" (Jun 13, 2026)
https://www.developersdigest.tech/blog/enterprise-ai-coding-budget-blowouts-2026
- Microsoft cancelling Claude Code across Experiences + Devices by June 30, 2026, directing engineers to GitHub Copilot CLI.
- Official reasoning cites integration advantages with Microsoft's repos, workflows, and security infrastructure; practical effect consolidates on Microsoft's own product.
- Context: Uber exhausted its entire 2026 AI-tools budget by April (four months in).

## digitalapplied.com — "The AI Cost Reckoning: Right-Sizing Model Spend"
https://www.digitalapplied.com/blog/ai-cost-reckoning-right-sizing-model-spend-2026
- Copilot Enterprise bills a flat per-seat rate; Claude Code charges a base seat fee + variable API token usage.
- "The exact per-seat figures are vendor-stated; the cost-structure contrast — flat versus consumption-based — is the independently confirmed part."
- "The lesson is not that one tool is better; it is that finance now prizes cost certainty over raw consumption."
- Gartner: >40% of agentic AI projects canceled by end of 2027 (cost overruns, unclear value, inadequate risk controls).

## GitHub Copilot pricing context (usagebox.com)
https://usagebox.com/articles/github-copilot-usage-based-billing-2026
- Copilot moved to usage-based billing June 1, 2026 (AI Credits, 1 credit = $0.01). Pro $10 (incl. $15 credits), Pro+ $39 (incl. $70), Max $100 (incl. $200). Business/Enterprise seats draw from org-pooled credit balance.
- Note: the "flat per-seat" contrast is becoming less clean as Copilot itself adds consumption-based credits — the model you pick decides burn rate.
