# FranklinCovey CIO Blaine Carter: buy -> build

Independent trade-press interview (diginomica), not a vendor case study. FranklinCovey is
a named public company (training/leadership; "7 Habits"). CIO Blaine Carter is named.

## Systems replaced / built in-house
1. Financial metrics "fiscal binder" project
   - Replaced a ~$50k/year software solution with a custom low-code system.
   - Removed manual binder circulation among a 10-person finance team.
   - Processing time cut from 30 days to "a few days."
   - ~$100k in avoided software spend + ~$10k reduced external audit fees.
2. RFP (request for proposal) management
   - Replaced dedicated RFP software with an AI-augmented low-code solution.
   - 85–90% time reduction in RFP completion.
   - $30k–$40k/year saved in software licenses.

## Why
Carter: "There's software we can buy that does much more than we need it to do, but
we're willing to pay the $50k a year to have this be digital because the process is so
cumbersome." Mantra: "Instead of customization, we use configuration... customization
leads to a lot of overhead." Success required "partnership with the business unit
feeling the pain."

## Tools
Make (low-code automation platform) as the primary build environment.
